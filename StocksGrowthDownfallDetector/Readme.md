# Stocks Uprise / Downfall Calculator 
# TECH USED- Python

This code was written to find out the behaviour of stocks from a csv file which can have infinity number of stock records. 
By behaviour i mean pick those particular stocks whose price have had an increase/decrease for n number of days continuously meaning if there was an increase after first day then its price went on increase till n days and vice versa .

# This helps in figuring out which companies are worth investing in .
